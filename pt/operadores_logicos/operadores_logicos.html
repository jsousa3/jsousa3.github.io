Operadores Lógicos

Operadores Lógicos

São suportados os operadores de negação, not, conjunção, and e disjunção or.

Estes operadores aplicam-se a valores booleanos.

Exemplos (no ficheiro exemplos_operadores_logicos.py)

# negação - not
print('not')
print(not True)
print(not False)
# conjunção - and
print('and')
print(True  and True)
print(True  and False)
print(False and True)
print(False and False)
# disjunção - or
print('or')
print(True  or True)
print(True  or False)
print(False or True)
print(False or False)

Funções booleanas

Fórmulas booleanas podem ser implementadas como funções Python.

Exemplos (no ficheiro funcao_booleana.py)

# f(p, q) = ((não p) ou q) e ((não q) ou p)

def f(p, q):

    resultado = ((not p) or q) and ((not q) or p)

    return resultado

print('((não p) ou q) e ((não q) ou p)')
print('Tabela de Verdade')
p = True
q = True
f_pq = f(p, q)
print(p, q, '|', f_pq)
p = True
q = False
f_pq = f(p, q)
print(p, q, '|', f_pq)
p = False
q = True
f_pq = f(p, q)
print(p, q, '|', f_pq)
p = False
q = False
f_pq = f(p, q)
print(p, q, '|', f_pq)
