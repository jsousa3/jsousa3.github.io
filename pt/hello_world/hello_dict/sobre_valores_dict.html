Sobre valores dict

Sobre valores dict

Os dicionários são valores do tipo dict. dict são as primeiras letras da palavra inglesa dictionary, que quer dizer "dicionário". Os dicionários são estruturas de dados. Servem para armazenar pares chave/valor. Fazem a associação entre chaves e os respetivos valores. Da mesma forma que um dicionário de uso corrente faz a associação entre as palavras e o seu significado.

Chaves

As chaves podem ser valores, como números inteiros, números decimais e strings ou ainda estruturas de dados imutáveis como tuplos.

Valores

Os valores podem ser de qualquer tipo.

Criar um dicionário

Para criar um dicionário colocam-se os pares chave/valor do cicionário, separados por vírgulas, entre chavetas. Os pares chave/valor são separados por dois pontos, :.

Exemplo

nomes_precos  = {'pão':1.23, 'água':7.08, 'café':3.00, 'sal':0.53}

Indexar

Para se obter o valor associado a uma chave usa-se a chave como se fosse um índice de uma lista. Entre parentsis retos.

Exemplo

print(nomes_precos['sal'])

Na verdade não se deveria chamar a este processo "indexar". Porque as chaves não são índices. Usa-se este termo apenas porque é útil fazer a analogia com as listas.

Alterar elemento

Os dicionários podem ser alterados. Para modificar um elemento de um dicionário usa-se o operador de atribuição. O sinal de igual. Do lado esquerdo coloca-se o dicionário indexado com a chave do elemento cujo valor se pretende alterar. Do lado direito o novo valor associado à chave em causa.

Exemplo

nomes_precos['sal'] = 0.55

Percorrer todos os pares chave/valor

O ciclo for, sobre um dicionário, permite obter todas as chaves dos pares que estão no dicionário. Com essas chaves podem obter-se os valores correspondentes.

Exemplo

for chave in nomes_precos:
    valor = nomes_precos[chave]
    print('----')
    print(chave)
    print(valor)

Erro

Se se tentar aceder ao valor de uma chave que não existe, o interpretador dá erro. Porque não sabe que valor retornar. Não existe nenhum valor.

Exemplo

print(nomes_precos['cacau'])

Verificar a existência de chave

A existência de uma dada chave por ser verificada com o operador de pertença in.

Exemplo

nome = 'cacau'
print(nome)
if nome in nomes_precos:
   preco = nomes_precos[nome]
   print(preco)
else:
   print('produto não existente')

Exemplo completo - no ficheiro exemplo_dicionario.py

# nomes_precos é um dicionário
# as chaves são nomes de produtos (strings)
# os valores são os preços dos produtos identiicados pelas chaves
nomes_precos  = {'pão':1.23, 'água':7.08, 'café':3.00, 'sal':0.53}

# valor associado à chave 'sal'
print(nomes_precos['sal'])

# alteração do valor associado à chave 'sal'
nomes_precos['sal'] = 0.55

# percorrer todos os pares chave/valor no dicionário
for chave in nomes_precos:
    valor = nomes_precos[chave]
    print('par chave/valor')
    print(chave)
    print(valor)

# testar se uma chave está no dicionário
nome = 'cacau'
print(nome)
if nome in nomes_precos:
   preco = nomes_precos[nome]
   print(preco)
else:
   print('produto não existente')

# a linha seguinte dá erro porque a chave 'cacau' não está no dicionário
print(nomes_precos['cacau'])
