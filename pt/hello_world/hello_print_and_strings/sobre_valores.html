Sobre valores (string)

'Hello World' é um valor. Os valores são sempre de algum tipo. O tipo do valor 'Hello World' é string. Mais precisamente o nome do tipo é str, que funciona como abreviação de string. A tradução para português de string, é corda, ou fio. A ideia é que uma string é uma cadeia de caracteres, ou uma sequência de carateres. Como a frase 'Hello World'.

Naturalmente que existem em Python outros tipos de valores, como por exemplo números, que veremos em seguida.
