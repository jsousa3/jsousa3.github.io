Sobre valores set

Sobre valores set

O tipo dos valores conjunto é set. A palavra inglesa set quer dizer "conjunto". Os conjuntos em Python são estruturas de dados. Servem para armazenar elementos. Os elementos de um conjunto podem ser de tipos diferentes.

Criar um conjunto

Para criar um conjunto colocam-se os elementos do conjunto, separados por vírgulas, entre chavetas.

Exemplo

# criar um conjunto
produtos_loja_1 = {'chocolate', 'água', 'café', 'chocolate'}

Face à ligação forte que existe entre a informática e a matemática (a base da infomática, no que se refere por exemplo à manipulação de informação representada no alfabeto binário, é matemática), é muto interessante constatar que a criação de um conjunto em Python usa exatamente a mesma notação usada em matemárica. Em geral, em matemática um conjunto escreve-se em extensão colocando os seus elementos separados por vírgulas, entre chavetas.

Diferença para os dicionários

As chavetas também são usadas para criar dicionários. A diferenças é que no caso dos dicionários, os seus elementos são pares chave/valor. Cada par chave/valor define-se com o valor da chave e o valor do valor, separados por dois pontos, :. No caso dos conjuntos os seus elementos são simplemente valores.

Exemplo

# criar outro conjunto
produtos_loja_2 = {'pão', 'água', 'café', 'pão'}
# criar um dicionário (só para comparação)
nomes_precos  = {'pão':1.23, 'água':7.08, 'café':3.00, 'sal':0.53}

Conjunto vazio

Se as chavetas são usadas tanto para dicionários como para conjuntos, coloca-se a questão de saber de que tipo é um valor criado com chavetas sem nenhum elemento. É um dicionário vazio ou um conjunto vazio? Os criadores da linguagem decidiram que as chavetas sem nenhum elemento são um dicionário vazio. Coloca-se então a questão de saber como se criar um conjunto vazio. É recorrendo à função built-in set. A função set cria um conjunto cujos elementos são os valores fornecidos como argumentos. Se se chamar a função sem argumentos, é criado um conjunto vazio.

Exemplo

# x é um dicionário vazio
x = {}
# y é um conjunto vazio
y = set()

Não há elementos repetidos

À semelhança do conceito matemático de conjunto, os conjunto em Python não têm elementos repetidos.

Exemplo

# os elementos repetidos não aparecem no output porque eles não
# existem em conjuntos
print(produtos_loja_1)
print(produtos_loja_2)

Operações sobre conjuntos

À semelhança do conceito matemático de conjunto, os conjuntos Python suportam as operações de pertença, inclusão (conjunto contido noutro), união, interseção, diferença e diferença exclusiva.

# relação de pertença
print('chocolate pertence ao conjunto produtos_loja_1?')
print('chocolate' in produtos_loja_1)
# relação de inclusão
print('o conjunto {água, café} está contido no conjunto produtos_loja_1?')
print({'água', 'café'} <= {'chocolate', 'água', 'café', 'chocolate'})
# conjunto de todos os produtos
uniao = produtos_loja_1 | produtos_loja_2
# produtos vendidos em ambas as lojas
intersecao = produtos_loja_1 & produtos_loja_2
# produtos exclusivos da loja 1
diferenca = produtos_loja_1 - produtos_loja_2
# produtos exclusivos na loja 1 ou na loja 2
diferenca_exclusiva = produtos_loja_1 ^ produtos_loja_2

Ordem dos elementos - indexação

À semelhança do conceito matemático de conjunto, a ordem dos elementos dos conjuntos Python não se define, não existe. Por isso os conjuntos Python não suportam indexação. A indexação dá erro. Não é permitida. O interpretador Python não sabe o que retorna quando se pretende obter o elemento de um conjunto no índice zero, ou um, porque a ordem dos elementos não se define.

# linha seguinte dá erro
print(produtos_loja_1[0])
