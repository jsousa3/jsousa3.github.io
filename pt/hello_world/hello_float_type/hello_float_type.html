Hello float type

Hello float type

Objetivo

Fazer output do tipo dos valores 1.23, 1.0 e 0.13.

Programa No ficheiro float_type.py

print(type(1.23))
print(type(1.0))
print(type(0.13))

Output

<class 'float'>
<class 'float'>
<class 'float'>

Observação

Em Python o tipo dos números decimais é float. São as primeiras letras de floating point, o que quer dizer vírgula flutuante, ou vígula deslizante. Floating point é o formato de representação de números décimais mais usado pelos processadores dos computadores. Recebeu este nome porque neste formato o separador décimal não está fixo. Desliza consoante o número em causa. É uma forma de otimizar a memória disponível nos processadores para representar números decimais. Permite representar números maiores e mais pequenos do que se o separador décimal estivesse fixo. A ideia é que quando os números são muito grandes as casa decimais não são relevantes e por isso pode-se representar os números usando menos memória para as casas deciamis e mais memória para a parte interira. Se por outro lado os números forem muito pequenos então a parte inteira não é relevante, é zero, e por isso pode-se representar os números usando mais memória para as casas deciamis e menos memória para a parte interira.

Exemplos

print(1.13 * 123456789)
print(1.13 / 123456789)

Output 

139506171.57
9.1530000832923e-09

Observação

O número 1.13 * 123456789 é muito grande, e por isso é representado apenas com duas casas décimais. Já o número 1.13 / 123456789 é muito pequeno e por isso é representado com 13 casas décimais.

Tal como já referimos, este formato de vírgula flutuante é muito usado nos procesaadoores dos computadores porque permite aumentar a gama de números decimais que é possível representar com uma quantidade de memória fixa.
