Hello ficheiros

Hello ficheiros

Considere a lista de preços, dispostos numa coluna:

1.23
7.08
3.00
0.53
2.01
4.44
9.99
2.50
0.20
7.70

No IDLE, faça CTRL n para criar um novo ficheiro. Copie estes preços para o novo ficheiro. E usando CTRL s, grave o ficheiro com o nome precos.txt.

A lista de preços está agora no ficheiro precos.txt. Um preço em cada linha.

Objetivo

Pretende-se escrever um programa que crie o ficheiro precos_altos.txt com todos os preços do ficheiro precos.txt que não começam por zero (neste contexto, os preços que começam por zero, como por exemplo, 0.53, não são considerados altos porque são inferiores à unidade).

Solução

Criar uma lista vazia. Abrir o ficheiro precos.txt em modo de leitura. Usando um ciclo while, ler cada uma das linhas do ficheiro precos.txt. Sempre que o primeiro elemento da string lida não seja zero, acrescentá-la à lista.

Abrir o ficheiro precos_altos.txt em modo de escrita. Usando um ciclo for escrever cada um dos elemntos da lista no ficheiro precos_altos.txt.

Programa - no ficheiro ficheiros_de_precos.py

lista_auxiliar = []

nome_do_ficheiro = 'precos.txt'
modo             = 'r'    # leitura. r de read
codificacao      = 'utf8' # isto é necessário em Windows
ficheiro_precos  = open(nome_do_ficheiro, modo, encoding=codificacao)

print('a ler o ficheiro de preços...')
for linha in ficheiro_precos:
    if linha[0] != '0':
       lista_auxiliar.append(linha)
print('a ler o ficheiro de preços... fim!')

ficheiro_precos.close()

nome_do_ficheiro      = 'precos_altos.txt'
modo                  = 'w'    # escrita. w de write
codificacao           = 'utf8' # isto é necessário em Windows
ficheiro_precos_altos = open(nome_do_ficheiro, modo, encoding=codificacao)

print('a escrever o ficheiro de preços altos...')
for linha in lista_auxiliar:
    ficheiro_precos_altos.write(linha)
print('a escrever o ficheiro de preços altos... fim!')

ficheiro_precos_altos.close()

Output

a ler ficheiro de preços...
a ler ficheiro de preços... fim!
a escrever ficheiro de preços altos...
a escrever ficheiro de preços altos... fim!

Output principal

O output principal deste programa está no ficheiro precos_altos.txt. Poderá abri-lo com o IDLE ppara verificar que tem este conteúdo.

1.23
7.08
3.00
2.01
4.44
9.99
2.50
7.70

Tal como era pretendido os preços 0.53 e 0.20 não estão no ficheiro precos_altos.txt.

Como funciona

Os ficheiros estão armazenados nos computadores. A função open permite aceder à posição exata da memória onde os ficheiros estão armazenados.

A função open tem três argumentos.

  • O nome do ficheiro. É uma string. Na forma mais simples de abetura de ficheiros os ficheiros têm que estar na mesma diretoria/pasta do programa que os está a abrir.

  • O modo de abertura. Os ficheiros podem ser abertos para leitira, 'r' (de read), ou para escrita, 'w' (de write). Existem ainda outros modos possíveis.

  • A codificação do ficheiro define o modo com a informação que está no ficheiro é interpretada. Note-se que um carater chinês pode ter mesma representação binário de uma letra do alfabeto ocidental. A forma como essa representação binária é interpretada é definida pela codificação indicada pelo programado usando o argumento encoding. A cofificação utf8 é uma codificação moderna que suporta todas as línguas do globo. Por isso é que atualmente é muitíssimo usada. Dizemos que é moderna por oposição a condificações mais antigas específicas para cada língua.

Os ficheiros podem ser lidos linha a linha, com um ciclo for:

for linha in ficheiro:

Pode escrever-se num ficheiro usando a função write. A função write só está disponível no contexto de um ficheiro, usando o operador ponto, .:

ficheiro.write(linha)

O número de ficheiros abertos simultaneamente num computador é limitado. Por isso os ficheiros que foram abertos tẽm que ser fechados usando a função close. à semelhança da função write, A função close só está disponível no contexto de um ficheiro, usando o operador ponto, .:

ficheiro.close()
