Funções

Funções

Uma função é uma parte de uma programa. Essa parte do programa é reutilizável. Pode ser usada em vários sítios.

Já vimos isso com a função print. Podemos chamar a função print em vários sítios.

E podemos criar as nossas próprias funções? Para fazerem o que nós quisermos? SIM, podemos.

Definição de funções

Para definir uma função usa-se a instrução def. São as letras iniciais da palavra inglesa define, que quer dizer "definir".

Uma função tem um nome, uma lista de argumentos, e um corpo. O corpo da função é a parte do progrma que é reutilizável.

Distingue-se o corpo da função, do restante programa onde a função está definitada, pela indentação. O corpo da função tem que estar indentado com 4 espaços em relação ao programa onde está definida.

Exemplo (no ficheiro definicao_de_funcao.py)

Escreva e execute este programa. Qual é o output?


def uma_funcao(mensagem, nome):

    print('**********')                 # esta linha faz parte da função
    print('**********', mensagem, nome) # esta também
    print('**********')                 # e esta também

print('Hello World') # mas esta linha não faz parte da função

Chamar funções

Para executar uma função chama-se a função. Para chamar a função usa-se o nome da função e fornece-se tantos valores quantos o número de argumentos da função, entre parentsis curvos e separados por vírgulas.

Exemplo (no ficheiro chamar_funcao.py)

Escreva e execute este programa.


def uma_funcao(mensagem, nome):

    print('**********')
    print('**********', mensagem, nome)
    print('**********')

uma_funcao('Olá',   'Beatriz')
uma_funcao('Olá',   'Manuel')
uma_funcao('Hello', 'John')
uma_funcao('Hello', 'Mary')

Retorno

Uma função pode retornar um valor. Para tal usa-se a instrução 'return' seguida do valor a retornar.

A instrução return interrompe a execução a função. Nada mais é executado no corpo da função depois ser ser executado o return.

A instrução return não é obrigatória. Mas uma função retorna sempre um valor. Quando não se usa a instrução return a função retorna None.

Exemplos (no ficheiro exemplos_retorno_funcao.py)

Escreva e execute este programa.


def dobro(um_numero):

    return 2 * um_numero

def ola(nome):

    print('Hello', nome)
    return
    print('Olá', nome)

def ola2(nome):

    print('Hello', nome)
    print('Olá', nome)

print(dobro(10))
print(ola('João'))
print(ola2('Maria'))

Argumentos

Os argumentos das funções são obrigatórios, isto é, se uma função é definida, por exemplo, com 2 argumentos, quando é chamada têm que se fornecer 2 valores.

Uma função pode não ter argumentos. Mas a chamada à função tem que ter na mesma os parenntsis curvos. Neste caso ficam vazios.

Exemplos (no ficheiro argumentos_funcao.py)

Escreva e execute este programa.


# esta função tem um argumento
def dobro(x):

    return 2 * x

# esta função tem três argumentos
def produto(x, y):

    return x * y

# esta função não tem argumentos
def hello():

    print('Hello World!')

print(dobro(10))
print(produto(10,20))
print(hello())
print(dobro(1, 2))
print(produto(3))
print(hello(4))

Contexto

Os argumentos da lista de argumentos da definição da função são variáveis locais à função. Isto quer dizer que não estão acessíveis fora da função e deixam de existir quando a execução da função termina.

As variáveis inicializadas dentro de uma função também são locais à função.

Exemplos (no ficheiro variaveis_locais.py)

Escreva e execute este programa.


def dobro(x):

    y = 2 * x

    return y

z = dobro(10)
print(z)
print(x)
print(y)
