Dicionários

Dicionários

Os dicionários são valores do tipo dict. dict são as primeiras letras da palavra inglesa dictionary, que quer dizer "dicionário". Os dicionários servem para armazenar pares chave/valor. Fazem a associação entre chaves e os respetivos valores. Da mesma forma que um dicionário de uso corrente faz a associação entre as palavras e os seus significados.

Chaves

As chaves podem ser valores, como números inteiros, números decimais e strings ou ainda estruturas de dados imutáveis como tuplos. Têm que ser únicas.

Valores

Os valores podem ser de qualquer tipo.

Criar dicionários

Para criar um dicionário colocam-se os pares chave/valor do cicionário, separados por vírgulas, entre chavetas. Os pares chave/valor são separados por dois pontos, :.

Exemplo

taxas_câmbio = {'USD':1.0527, 'JPY':159.74, 'GBP':0.82555, 'CHF':0.9267, 'BRL':6.3811}
print(taxas_câmbio)

Número de elementos dos dicionários

Exemplo

print(len(taxas_câmbio))

Criar dicionário vazio

Exemplo

dicionário_vazio = {}
print(len(dicionário_vazio))

Obter valor (indexar)

Para se obter o valor associado a uma chave usa-se a chave como se fosse um índice de uma lista. Entre parentsis retos.

Exemplo

print(taxas_câmbio['GBP'])

Na verdade não se deveria chamar a este processo "indexar". Porque as chaves não são índices. Usa-se este termo apenas porque é útil fazer a analogia com as listas.

Neste contexto pode entender-se os diciońarios como extensões das listas de forma a poder usar índices que não são números inteiror.

Alterar par chave/valor

Os dicionários podem ser alterados.

Exemplo

taxas_câmbio['GBP'] = 0.9
print(taxas_câmbio)

Adicinar par chave/valor

Exemplo

taxas_câmbio['xpto'] = 0.0
print(taxas_câmbio)

Remover par chave/valor

Exemplo

taxas_câmbio.pop('xpto')
print(taxas_câmbio)

Percorrer todos os pares chave/valor

O ciclo for, sobre um dicionário, permite obter todas as chaves dos pares que estão no dicionário. Com essas chaves podem obter-se os valores correspondentes.

Exemplo

for chave in taxas_câmbio:
    valor = taxas_câmbio[chave]
    print('----')
    print(chave)
    print(valor)

Erro

Se se tentar aceder ao valor de uma chave que não existe, o interpretador dá erro. Porque não sabe que valor retornar. Não existe nenhum valor.

Exemplo

print(taxas_câmbio['EUR'])

Verificar a existência de chave

A existência de uma dada chave pode ser verificada com o operador de pertença in.

Exemplo

código = 'EUR' # altere este código para 'GBP' e volte a executar
print(código)
if código in taxas_câmbio:
   taxa = taxas_câmbio[taxa]
   print(taxa)
else:
   print('código não existente')

Obter todas as chaves

Exemplo

chaves = taxas_câmbio.keys()
print(chaves)

Obter todos os valores

Exemplo

valores = taxas_câmbio.values()
print(valores)
