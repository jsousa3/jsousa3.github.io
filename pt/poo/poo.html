POO - Programação Orientada por Objetos

Tal como já foi referido, para além dos tipos de valores já existentes em Python, a Programação Orientada por Objetos permite criar novos tipos.

  • Os valores desses novos tipos chamam-se objetos.

  • A "receita" que permite criar objetos é a classe (por isso antes de criar objetos é preciso criar a classe que permite criar esses objetos).

  • Diz-se que um objeto é uma instância de uma classe.

  • Os objetos têm atributos.

  • O acesso aos atributos faz-se com o operador . (ponto).

  • Há atributos que são valores. Servem para armazenar dados. A estes atributos chamam-se membros.

  • Há atributos que são funções. Servem para efetuar operações sobre os dados. A estes atributos chamam-se métodos.

  • Por convenção a variável self representa o próprio objeto.

  • Há um método especial, o construtor, que é executado para criar e inicializar os objetos. O nome do construtor é __init__.

  • O primeiro argumento de todos os métodos é self.

  • Os objetos são mutáveis, na medida em que os seus atributos podem ser alterados.
