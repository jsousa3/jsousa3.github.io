POO print de objetos

print de objetos

O que acontece quando se faz print de um obejto?

Exemplo (no ficheiro print_objeto.py)

Escreva e execute este programa. Qual é o output?


class Pessoa:

    def __init__(self, peso, altura):

        self.peso   = peso
        self.altura = altura

    # método para calcular o IMC
    def f(self):

        return self.peso / (self.altura * self.altura)

peso_1   = 80  # em Kg
altura_1 = 1.8 # em metros
pessoa_1 = Pessoa(peso_1, altura_1)

peso_2   = 70  # em Kg
altura_2 = 1.7 # em metros
pessoa_2 = Pessoa(peso_2, altura_2)

print('*** pessoa_1 ***')
print(pessoa_1)
print('*** pessoa_2 ***')
print(pessoa_2)

O output é este:

*** pessoa_1 ***
<__main__.Pessoa object at 0x7f650bdc1b50>
*** pessoa_2 ***
<__main__.Pessoa object at 0x7f650bdc1fd0>

É muito pouco útil. Diz-nos que o objeto pessoa_1 é uma instância da classe Pessoa que está no endereço de memória 0x7f650bdc1b50. (A classe Pessoa está definida no contexto do programa principal, que se chama __main__.) Já o objeto pessoa_2 está no endereço 0x7f650bdc1fd0. Não só são detalhes com pouco interesse, como variam de computador para computador porque a memória existente e a memória disponível é diferente de computador para computador.

Pergunta

Será possível redefinir este output para que o print seja controlado pelo programador?

Resposta

Sim, (re)definindo, por exemplo, o método especial __repr__.

repr são as 4 primeiras letras de representation, que, em português, quer dizer "representação".

Por convençao o método __repr__ deve retornar uma string com uma representação do objeto.

Se o método __repr__ estiver definido numa classe, quando se faz print de uma instância dessa classe (um objeto do tipo dessa classe), o interpretador Python obtem a representação em string do objeto usando esse método e passa o resultado obtido à função print.

Exemplo (no ficheiro print_objeto_com_repr.py)

Escreva e execute este programa. Qual é o output?


class Pessoa:

    def __init__(self, peso, altura):

        self.peso   = peso
        self.altura = altura

    def __repr__(self):

        return 'Pessoa(peso = ' + str(self.peso) + ', altura = ' \
            + str(self.altura) + ')'

    # método para calcular o IMC
    def f(self):

        return self.peso / (self.altura * self.altura)

peso_1   = 80  # em Kg
altura_1 = 1.8 # em metros
pessoa_1 = Pessoa(peso_1, altura_1)

peso_2   = 70  # em Kg
altura_2 = 1.7 # em metros
pessoa_2 = Pessoa(peso_2, altura_2)

print('*** pessoa_1 ***')
print(pessoa_1)
print('*** pessoa_2 ***')
print(pessoa_2)

O output agora é este:

*** pessoa_1 ***
Pessoa(peso = 80, altura = 1.8)
*** pessoa_2 ***
Pessoa(peso = 70, altura = 1.7)

É inteiramente controlado pelo programador.
