Alterar

Alterar listas

Os elementos armazenados nas listas podem ser alterados. As listas são alteráveis. O termo usado em contexto Python é que as listas são mutáveis.

Alterar usando o índice do elemento

Pode-se alterar cada um dos elementos de uma lista através do índice respetivo.

Exemplos (no ficheiro listas_exemplos_alterar_usando_indice.py)

produto = ['Farinha', 1.23, 10, True]
print(produto)
produto[0] = 'Arroz'
print(produto)
produto[1] = 0.80
print(produto)
produto[2] = 5
print(produto)
produto[3] = False
print(produto)

Output

['Farinha', 1.23, 10, True]
['Arroz', 1.23, 10, True]
['Arroz', 0.8, 10, True]
['Arroz', 0.8, 5, True]
['Arroz', 0.8, 5, False]

Alterar usando funções

Podem acrescentar-se elementos ao fim das listas usando a função append. Podem inserir-se elementos num dado índice usando a função insert. Pode remover-se um elemento através do seu índice usando a função pop. A função pop retorna o elemento removido.

As funções append, insert e pop só existem no contexto das listas. Para serem chamadas usa-se o operador ponto . a seguir à variável que armazena uma lista, seguido da chamada à função pretendida. A função é executada sobre a lista armazenada na variável usada. Trata-se de um mecanismo relativo à programação orientada por objetos que veremos mais à frente.

Exemplos (no ficheiro listas_exemplos_alterar_usando_funcoes.py)

lista_de_preços = [97, 70, 15, 38, 92, 30, 86, 68, 6, 80]
print(lista_de_preços)
# acrescentar um preço no final da lista
lista_de_preços.append(100)
print(lista_de_preços)
# inserir um preço no terceiro elemento
lista_de_preços.insert(2, 200)
print(lista_de_preços)
# remover o primeiro preço
preço_removido = lista_de_preços.pop(0)
print(lista_de_preços)
print(preço_removido)

Output

[97, 70, 15, 38, 92, 30, 86, 68, 6, 80]
[97, 70, 15, 38, 92, 30, 86, 68, 6, 80, 100]
[97, 70, 200, 15, 38, 92, 30, 86, 68, 6, 80, 100]
[70, 200, 15, 38, 92, 30, 86, 68, 6, 80, 100]
97
