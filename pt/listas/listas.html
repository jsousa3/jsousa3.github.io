Listas

Listas

Para além dos valores elementares como as strings, os números inteiros, os números decimais, ou os valores booleanos, existem em Python outros valores que permitem armazenar informação. É o caso das listas. O tipo das listas é list.

Criação

Uma lista é uma sequência ordenada de valores. Aos valores das listas também se chamam elementos. Os valores numa lista podem ser de tipos diferentes. Isto é, uma lista pode ter um número inteiro, uma string, e um booleano, por exemplo.

Exemplos (no ficheiro exemplo_lista_criacao.py)

# uma lista com um número inteiro, uma string,
# e um boleano
uma_lista = [10, 'hello', False]
print(uma_lista)

Listas de listas

Os tipos dos valores nas listas podem ser quaisquer, até mesmo do tipo list. Isto é, uma lista pode ter elementos que são eles próprios listas.

Exemplos (no ficheiro exemplo_lista_com_listas.py)

# uma lista cujos elementos são listas
uma_lista = [[1, 2, 3], ['a', 'b', 'c'], [True, 'hello', -2.0]]
print(uma_lista)

Indexação

Os valores nas listas podem ser obtidos pelo seu índice na sequência. Chama-se a isto indexação da lista, ou indexar a lista. Os índices das listas são números inteiros e iniciam-se em zero. Quer isto dizer que o índice do primeiro valor é zero, o do segundo é um, o do terceiro é dois, e assim por diante.

Exemplos (no ficheiro exemplos_lista_indexacao.py)

uma_lista = [10, 'hello', False]
print(uma_lista)
print('primeiro elemento')
print(uma_lista[0])
print('segundo elemento')
print(uma_lista[1])
print('terceiro elemento')
print(uma_lista[2])

Erro - indexação de elemento que não existe

Se se indexar uma lista num elemento que não existe, o interpretador Python não inventa nenhum valor pré-definido e dá erro.

Exemplo (no ficheiro exemplo_lista_indexacao_erro.py)

uma_lista = [10, 'hello', False]
# a próxima linha de código dá erro porque o quarto elemento da lista
# uma_lista não existe.
print(uma_lista[3])

Indexação de listas de listas

Se um elemento de uma lista, obtido por indexação, é ele próprio uma lista, ele pode também ser indexado. A dupla indexação pode ser efetuada numa única instrução.

Exemplos (no ficheiro exemplo_lista_com_listas_indexacao.py)

# uma lista cujos elementos são listas
uma_lista = [[1, 2, 3], ['a', 'b', 'c'], [True, 'hello', -2.0]]
print(uma_lista)
auxiliar = uma_lista[1] # este elemento é a lista ['a', 'b', 'c']
print(auxiliar)
auxiliar = auxiliar[2] # este elemento é a string 'c'
print(auxiliar)
# a dupla indexação pode ser efetuada numa única instrução
# uma_lista[1] é uma lista que pode ser indexada para obter-se o
# terceiro elemento, no índice 2
auxiliar = uma_lista[1][2]
print(auxiliar)

Alteração dos elementos das listas

As listas são mutáveis, o que quer dizer que podem ser alteradas.

Pode-se alterar cada um dos elementos de uma lista através do índice respetivo.

Exemplos (no ficheiro exemplos_lista_alterar_usando_indice.py)

uma_lista = [10, 'hello', False]
print(uma_lista)
uma_lista[0] = 100
print(uma_lista)
uma_lista[1] = 200
print(uma_lista)
uma_lista[2] = 300
print(uma_lista)

Pode acrescentar-se elementos ao fim das listas usando a função append. Pode inserir-se elementos num dado índice usando a função insert. Pode remover-se elementos através do índice usando a função pop. A função pop retorna o elemento removido.

As funções append, insert e pop só existem no contexto das listas. Para serem chamadas usa-se o operador ponto . a seguir à variável que armazena uma lista, seguido da chamada à função pretendida. A função é executada sobre a lista armazenada na variável usada.

Exemplos (no ficheiro exemplos_lista_alterar_usando_funcoes.py)

uma_lista = [10, 'hello', False]
print(uma_lista)
# acrescentar um float no final da lista
uma_lista.append(2.0)
print(uma_lista)
# inserir uma string no terceiro elemento
uma_lista.insert(2, 'world')
print(uma_lista)
# remover o primeiro elemento
elemento_removido = uma_lista.pop(0)
print(uma_lista)
print(elemento_removido)

Número de elementos de listas

Pode obter-se o número de elementos de uma lista usando a função buitl-in len. São as três primeiras letras da palavra inglesa length, que quer dizer comprimento. Neste contexto, a função len retorna o número de elementos da lista, ou o comprimento da lista.

Exemplos (no ficheiro exemplos_lista_len.py)

uma_lista = [10, 'hello', False]
print(uma_lista)
# número de elemntos da lista
print(len(uma_lista))
