Listas

Para além dos valores elementares como as strings, os números inteiros, os números decimais, ou os valores booleanos, existem em Python outros valores que permitem armazenar informação. É o caso das listas.

Uma lista é uma sequência ordenada de valores. Aos valores das listas também se chamam elementos. Os valores numa lista podem ser de tipos diferentes. Isto é, uma lista pode ter um número inteiro, uma string, e até outra lista, por exemplo.

Os valores nas listas podem ser obtidos pelo seu índice na sequência. Chama-se a isto indexação da lista, ou indexar a lista. Os índices das listas iniciam-se em zero. Quer isto dizer que o índice do primeiro valor é zero, o do segundo é um, o do terceiro é dois, e assim por diante.

As listas são mutáveis, o que quer dizer que podem ser alteradas. Pode-se acrescentar elementos ao fim das listas usando a função append. Pode-se inserir elementos num dado índice usando a função insert. Pode-se remover elementos através do índice usando a função pop. A função pop retorna o elemento removido.

As funções append, insert e pop só existem no contexto das listas. Por isso têm que ser chamadas a partir das próprias listas, usando o operador ponto ..

Pode obter-se o número de elementos de uma lista usando a função buitl-in len. São as três primeiras letras da palavra inglesa length, que quer dizer comprimento. Neste contexto, a função len retorna o número de elementos da lista, ou o comprimentos da lista.

Indexar uma lista num elemento que não existe, dá erro.

Exemplos (no ficheiro exemplos_lista.py)

# uma lista com um número inteiro, uma string,
# e um boleano
uma_lista = [10, 'hello', False]
print(uma_lista)
print('primeiro elemento')
print(uma_lista[0])
print('segundo elemento')
print(uma_lista[1])
print('terceiro elemento')
print(uma_lista[2])
print('acrescentar um float no fim')
uma_lista.append(2.0)
print(uma_lista)
print('inserir uma string no terceiro elemento')
uma_lista.insert(2, 'world')
print(uma_lista)
print('remover o primeiro elemento')
uma_lista.pop(0)
print(uma_lista)
print('número de elemntos da lista')
print(len(uma_lista))
# a próxima linha dá erro, porque o elemento quinto não existe
print(uma_lista[4])
