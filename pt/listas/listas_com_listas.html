Listas com listas

Listas com listas

Os elementos das listas podem ser de qualquer tipo, incluindo do tipo listas. Isto quer dizer que as listas podem armazenar outras listas.

Exemplos (no ficheiro listas_exemplos_com_listas.py)

# uma lista com listas
tabela = [
    [0, 2, 0, 0, 0],
    [3, 0, 7, 0, 9],
    [1, 0, 5, 0, 1]
]
print(tabela)

Output

[[0, 2, 0, 0, 0], [3, 0, 7, 0, 9], [1, 0, 5, 0, 1]]

Indexeção de listas com listas

A indexação de listas com listas pode ser efetuada em cadeia, isto é, se o resultado da indexação de uma lista é uma lista, então essa indexação pode ser seguida de nova indexação.

Exemplos (no ficheiro listas_exemplos_com_listas_indexacao.py)

# uma lista com listas
tabela = [
    [0, 2, 0, 0, 0],
    [3, 0, 7, 0, 9],
    [1, 0, 5, 0, 1]
]
print(tabela)
print(tabela[0])
print(tabela[0][0])
print(tabela)
print(tabela[1])
print(tabela[1][1])
print(tabela)
print(tabela[2])
print(tabela[2][4])
print(tabela)
print(tabela[-1])
print(tabela[-1][-1])

Output

[[0, 2, 0, 0, 0], [3, 0, 7, 0, 9], [1, 0, 5, 0, 1]]

0
[[0, 2, 0, 0, 0], [3, 0, 7, 0, 9], [1, 0, 5, 0, 1]]

0
[[0, 2, 0, 0, 0], [3, 0, 7, 0, 9], [1, 0, 5, 0, 1]]
[1, 0, 5, 0, 1]

[[0, 2, 0, 0, 0], [3, 0, 7, 0, 9], [1, 0, 5, 0, 1]]
[1, 0, 5, 0, 1]

Indexeção de listas com listas com ciclos for

Todas as listas numa estrutura de listas com listas podem ser indexadas com ciclos for. Porque são listas.

Exemplos (no ficheiro listas_exemplos_com_listas_ciclo_for.py)

# uma lista com listas
tabela = [
    [0, 2, 0, 0, 0],
    [3, 0, 7, 0, 9],
    [1, 0, 5, 0, 1]
]
print(tabela)

for linha in tabela:
    print('linha =', linha)
    for valor in linha:
        print('valor =', valor)

# agora com índices
print('''
#############################
##### agora com índices #####
#############################
''')

for índice_linha in range(len(tabela)):
    linha = tabela[índice_linha ]
    print('linha =', linha)
    for índice_coluna in range(len(linha)):
        valor = linha[índice_coluna]
        print('valor =', valor)
