Operadores Aritméticos

Operadores Aritméticos

São suportadas as operações de adição +, subtração -, multiplicação *, divisão /, exponenciação **, divisão inteira //, e resto da divisão inteira %.

Em geral, os operadores +, -, *, /, **, // e %, aplicam-se apenas a números, isto é, tanto do lado esquerdo do operador, como do lado direito, têm que estar números.

Exemplos (no ficheiro exemplos_operadores.py)

# adição
print(7 + 2)
# subtração
print(7 - 2)
# multiplicação
print(7 * 2)
# divisão (o resultado é sempre um float)
print(7 / 2)
# exponenciação
print(7 ** 2)
# divisão inteira (o resultado é sempre um inteiro)
print(7 // 2)
# resto da divisão inteira
print(7 % 2)
# esta linha dá erro
# porque não faz sentido dividir a palavra 'hello' pela palavra 'world'
# para evitar o erro esta linha tem que ser apagada ou comentada
print('hello' / 'world')
