f-strings

f-strings

f-string é a designação abreviada para formatted string literal, que quer dizer string formatada.

Criação

Para criar-se uma f-string, coloca-se um f antes do limitador inicial da string. Qualquer dos limitadores pode ser usado (aspas, plicas, aspas triplas ou plicas triplas).

Programa (no ficheiro fstrings_criacao.py)

Escreva e execute este programa. Confirme o seu output.

s1 = f'isto é uma f-string'
s2 = f'''e isto
é outra
f-string'''
print(s1)
print(s2)

Output

isto é uma f-string
e isto
é outra
f-string

Substituição de variáveis

As variáveis incluidas entre chavetas são substitídas pelos seus valores.

Programa (no ficheiro fstrings_substituicao.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 10
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda}
preço:         {preço}
desconto:      {desconto_em_percentagem}%
preço a pagar: {preço_a_pagar}
'''
print(s1)

Output


moeda:         EUR
preço:         20
desconto:      10%
preço a pagar: 18

Muito útil - incluir o nome das variáveis usadas

Muitas vezes, quando se está a verificar o funcionamento de um programa, faz-se print dos nomes das variáveis e dos seus valores. Isso obriga a ter que se escrever duas vezes o nome das variáveis. Com f-strings só se tem que escrever uma vez o nome das variáveis.

Programa (no ficheiro fstrings_nomes_variaveis.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 0.1
preço_a_pagar           = 18

# sem f-strings
print('moeda =', moeda)
print('preço =', preço)
print('desconto_em_percentagem =', desconto_em_percentagem)
print('preço_a_pagar =', preço_a_pagar)

# com f-strings (esta versão dá muito menos trabalho a escrever)
print(f'{moeda = }')
print(f'{preço = }')
print(f'{desconto_em_percentagem = }')
print(f'{preço_a_pagar = }')

Output

moeda = EUR
preço = 20
desconto_em_percentagem = 0.1
preço_a_pagar = 18
moeda = 'EUR'
preço = 20
desconto_em_percentagem = 0.1
preço_a_pagar = 18

Formatação - número de casas decimais (só para números)

Pode formatar-se cada variável incluída dentro das chavetas, colocando : a seguir ao nome da variável, e acrescentado a formatação pretendida a seguir aos :. Para formatar-se o número de casas decimais de um número, acrescenta-se a seguir ao :, um ., o número de casas decimais pretendido, e um f (o f, de fixed-point notation, indica que se pretende usar um número fixo de casas decimais).

Programa (no ficheiro fstrings_casas_decimais.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 10
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda}
preço:         {preço:.1f}
desconto:      {desconto_em_percentagem:.5f}%
preço a pagar: {preço_a_pagar:.2f}
'''
print(s1)

Output


moeda:         EUR
preço:         20.0
desconto:      10.00000%
preço a pagar: 18.00

Formatação - tamanho ocupado

Pode indicar-se qual o número de carateres que o valor formatada deverá ocupar. Para tal acrescenta-se esse número de carateres depois do : (e antes da formatação das casas decimais caso exista).

Programa (no ficheiro fstrings_tamanho_ocupado.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 10
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda:10}
preço:         {preço:10.1f}
desconto:      {desconto_em_percentagem:10.5f}%
preço a pagar: {preço_a_pagar:10.2f}
'''
print(s1)

Output


moeda:         EUR       
preço:               20.0
desconto:        10.00000%
preço a pagar:      18.00

Formatação - zeros à esquerda (só para números)

Para acrescentarem-se zeros à esquerda precede-se o número de carateres a ocupar com um zero.

Programa (no ficheiro fstrings_zeros_esquerda.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 10
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda:10}
preço:         {preço:010.1f}
desconto:      {desconto_em_percentagem:010.5f}%
preço a pagar: {preço_a_pagar:010.2f}
'''
print(s1)

Output


moeda:         EUR       
preço:         00000020.0
desconto:      0010.00000%
preço a pagar: 0000018.00

Formatação - alinhamento

Para alinhar-se o valor formatado no espaço disponível, precede-se toda a formatação com ^ para centrar, com < para linha à esquerda, e com > para alinha à diretita.

Programa (no ficheiro fstrings_alinhamento.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 10
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda:^10}
preço:         {preço:>10.1f}
desconto:      {desconto_em_percentagem:<10.5f}%
preço a pagar: {preço_a_pagar:^10.2f}
'''
print(s1)

Output


moeda:            EUR    
preço:               20.0
desconto:      10.00000  %
preço a pagar:   18.00

Formatação - percentagem junto ao número (só para números)

No caso de se querer incluir o símbolo de percentagem na formação (útil para ele não ficar separado do número no caso do alinhamento à esquerda e ao centro) inclui-se o símbolo de percentagem % no final da própria formatação (dentro das chavetas). Considera-se valores a formatar como percentagem estão en tre zero e um, e por isso são multiplicados por cem pela própria formatação.

Programa (no ficheiro fstrings_percentagem.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 0.1
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda:^10}
preço:         {preço:>10.1f}
desconto:      {desconto_em_percentagem:<10.5%}
preço a pagar: {preço_a_pagar:^10.2f}
'''
print(s1)

Output


moeda:            EUR    
preço:               20.0
desconto:      10.00000% 
preço a pagar:   18.00

Formatação - incluir sinal (só para números)

Para inclui-se o sinal dos números inclu-se o sinal + depois do alinhamento.

Programa (no ficheiro fstrings_sinal.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 0.1
preço_a_pagar           = 18
s1 = f'''
moeda:         {moeda:^10}
preço:         {preço:>+10.1f}
desconto:      {desconto_em_percentagem:<+10.5%}
preço a pagar: {preço_a_pagar:^+10.2f}
'''
print(s1)

Output


moeda:            EUR    
preço:              +20.0
desconto:      +10.00000%
preço a pagar:   +18.00

Curiosidade - incluir chavetas

Para incluir chavetas numa f-string duplicam-se as chavetas.

Programa (no ficheiro fstrings_chavetas.py)

Escreva e execute este programa. Confirme o seu output.

moeda                   = 'EUR'
preço                   = 20
desconto_em_percentagem = 0.1
preço_a_pagar           = 18
s1 = f'''
{{moeda}}:         {moeda:^10}
{{preço}}:         {preço:>+10.1f}
{{desconto}}:      {desconto_em_percentagem:<+10.5%}
{{preço a pagar}}: {preço_a_pagar:^+10.2f}
'''
print(s1)

Output


{moeda}:            EUR    
{preço}:              +20.0
{desconto}:      +10.00000%
{preço a pagar}:   +18.00
