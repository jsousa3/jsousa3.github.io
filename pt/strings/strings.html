str

Strings

As strings são sequências de caracteres (por exemlo, frases ou textos). Os carateres podem ser letras, algarismos, ou quaisquer outros como espaços ou mesmo fins de linha.

O nome do tipo é str. São as três primeiras letras da palavra inglesa string.

As strings podem conter o caracter espaço. Por exemplo, a string 'Hello World' tem um espaço a separar as duas palavras. Por isso tem que se usar um limitador para definir o princípio e o fim das strings.

O limitador pode ser aspas " ou pelicas ', ou aspas triplas """, ou pelicas triplas '''. Uma string limitada por aspas pode ter pelicas. Uma string* limitada por pelicas pode ter aspas. Se a string não tiver aspas nem pelicas, a escolha entre usar aspas ou pelicas como limitador é indiferente.

A diferença entre aspas ou pelicas, simples e triplas, é que quando se usam triplas o texto pode ocupar várias linhas. É muito útil para textos grandes.

No teclado português como as aspas só estão acessíveis através da combinação shift + ", como se fosse uma letra maiúscula, e como as pelicas estão acessíveis diretamente na tecla do ponto de interrogação, sugere-se que se usem as pelicas. É mais rápido.

Exemplos (no ficheiro exemplos_strings.py)

# uma string definida com aspas
print("uma string")
# uma string definida com pelicas
print('#####') # um separador
# outra string definida com pelicas
print('outra string')
print('#####') # um separador
# as strings entre aspas podem ter pelicas
print("esta string tem pelicas aqui ' e aqui '")
print('#####') # um separador
# as strings entre pelicas podem ter aspas
print('esta string tem aspas aqui " e aqui "')
print('#####') # um separador
# uma string que ocupa 2 linhas definida com aspas
print("""esta é a primeira linha
e esta é a segunda""")
print('#####') # um separador
# uma string que ocupa 3 linhas definida com pelicas
print('''esta é a primeira linha
e esta é a segunda
e esta é a terceira''')

Output

uma string
#####
outra string
#####
esta string tem pelicas aqui ' e aqui '
#####
esta string tem aspas aqui " e aqui "
#####
esta é a primeira linha
e esta é a segunda
#####
esta é a primeira linha
e esta é a segunda
e esta é a terceira

Concatenação de strings

Uma string pode ser acrescentada a outra, criado-se uma terceira string. A esta operação chama-se concatenação de strings. O operador de concatenação é o sinal mais +.

Exemplo (no ficheiro concatenacao_strings.py)

print('"esta string' + 'foi concatenada com esta"')

Output

"esta stringfoi concatenada com esta"

Observação

A concatenação de strings não acrescenta nada às strings concatenadas. Nenhum separardor. Concatena as strings tal como elas são.
