Ficheiros

Ficheiros

É possível escrever e ler ficheiros a partir de programas Python.

Para se escrever ou ler um ficheiro, em primeiro lugar tem que se abrir o ficheiro. Para se abrir um ficheiro usa-se a função built-in open.

O primeiro argumento da função open é uma string com o nome do ficheiro que se vai abrir.

O segundo argumento é o modo em que o ficheiro é aberto. Os modos possíveis são:

Modo Descrição
'r' open for reading (default)
'w' open for writing, truncating the file first
'x' open for exclusive creation, failing if the file already exists
'a' open for writing, appending to the end of the file if it exists
'b' binary mode
't' text mode (default)
'+' open a disk file for updating (reading and writing)

Uma explicação sucinta sobre a diferença entre os modos de texto e binário. O que está armazenado nos ficheiros são bits.

Se sabemos que o ficheiro contem texto, por exemplo um programa Pyhton, queremos que os bits do ficheiro sejam convertidos diretamente nas letras correspondentes. Neste caso usamos o modo 't'.

Se sabemos que o ficheiro não contem texto, por exemplo é um ficheiro com uma imagem, não queremos que os bits do ficheiro sejam convertidos diretamente nas letras, porque isso não faria sentido. Neste caso usamos o modo 'b'.

Estes modos podem ser combinados, por exemplo, wb abre o ficheiro para escrita em modo binário.

A função open retorna o ficheiro aberto.

Escrever texto num ficheiro

Para se escrever num ficheiro, abre-se o ficheiro em modo 'w'. Se o ficheiro não existir, é criado. Se já existir, é recriado, isto é perde-se tudo o que lá estava armazenado anteriormente.

Depois de aberto o ficheiro, usa-se a função write. A função write só existe no contexto do ficheiro retornado pela função open. Para ser chamada tem que se usar o operador ponto . (tal como, por exemplo, para chamar a função append das listas).

Exemplo (no ficheiro ficheiro_escrever.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'w' # não é necessário usar 'wt' porque t é o modo
                    # pré-definido (default)
ficheiro = open(nome_ficheiro, modo)
uma_string_com_3_linhas = '''exemplo de
texto com
3 linhas'''
ficheiro.write(uma_string_com_3_linhas)
ficheiro.close()

O número de ficheiros abertos num computador é limitado. Depende do sistema operativo (Windos, ou MaxOS, ou GNU/Linux). Para não de deixar para trás ficheiros abertos desnecessariamente, cada ficheiro aberto deve ser fechado.

Fechar um ficheiro

Para fechar um ficheiro usa-se a função close (com operador ponto .).

No exemplo anterior já se fechou o ficheiro aberto.

Ler um ficheiro todo de uma vez

Para ler um ficheiro abre-se o ficheiro em modo 'r'. Para ler o seu conteudo todo de uma vez usa-se a função read (com operador ponto .).

Exemplo (no ficheiro ficheiro_ler_tudo.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'r'
ficheiro = open(nome_ficheiro, modo)
tudo = ficheiro.read() # ler o ficheiro todo de uma vez
ficheiro.close()
print(tudo)

Ler um ficheiro linha a linha com um ciclo for

Para ler um ficheiro linha a linha pode usar-se um ciclo for.

Note-se que as linhas lidas têm sempre o caracter de fim de linha no final. Porque ele está no ficheiro.

Exemplo (no ficheiro ficheiro_ler_com_ciclo_for.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'r'
ficheiro = open(nome_ficheiro, modo)
n_linha = 1
for linha in ficheiro:
    print('linha', n_linha, ' = "', linha, '"')
    n_linha = n_linha + 1
ficheiro.close()

Ler um ficheiro linha a linha com readline

Para ler um ficheiro linha a linha pode usar-se a função readline (com o operador ponto .) e um ciclo while.

Exemplo (no ficheiro ficheiro_ler_com_readline.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'r'
ficheiro = open(nome_ficheiro, modo)
n_linha = 1
linha = ficheiro.readline()
while linha != '':
    print('linha', n_linha, ' = "', linha, '"')
    n_linha = n_linha + 1
    linha = ficheiro.readline()
ficheiro.close()

Acrescentar texto a um ficheiro

Para acrescentar informação a um ficheiro usa-se o modo append 'a', e a função write.

Exemplo (no ficheiro ficheiro_append.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'a'
ficheiro = open(nome_ficheiro, modo)
uma_string = '''
e esta é uma linha final
(o texto total fica com 5 linhas)'''
ficheiro.write(uma_string)
ficheiro.close()

Ler um ficheiro em modo binário

Quando se lê o ficheiro em modo binário, cada byte lido (sequência de 8 bits) é um número de 0 a 255.

Exemplo (no ficheiro ficheiro_ler_binario.py)

nome_ficheiro = 'exemplo.txt'
modo          = 'rb'
ficheiro = open(nome_ficheiro, modo)
bytes = ficheiro.read(1)
while len(bytes) == 1:
    print('byte = ', bytes[0], ' (conversão para letra = ', chr(bytes[0]), ')')
    bytes = ficheiro.read(1)
ficheiro.close()
