Exercício 7 - Existe desconto grande

Exercício 7 - Existe pelo menos um desconto muito grande (maior que 80%)

Considere a lista de descontos que se segue:

lista_de_descontos = [10, 15, 95, 20, 10, 10, 30, 20, 10, 5]

Escreve um programa que faça print('existe pelo menos um desconto maior que 80%'), se existir pelo menos um desconto maior que 80% na lista de descontos. E que faça print('não existe nenhum desconto maior que 80%'), caso contrário.

A solução deverá poder ser aplicada a listas com milhares ou milhões de descontos (aplicados a milhares ou milhões de produtos).

Solução

Usar um ciclo for.

Programa (no ficheiro exercicio_7_existe_desconto_grande.py)

Escreva e execute este programa. Confirme o seu output.


lista_de_descontos = [10, 15, 95, 20, 10, 10, 30, 20, 10, 5]

desconto_grande = 80

existe = False
for desconto in lista_de_descontos:

    #print(desconto) # descomente este print só para confirmar quantas
                     # vezes este ciclo for é executado
    if desconto > desconto_grande:
        existe = True

if existe == True:
    print('existe pelo menos um desconto maior que 80%')
else:
    print('não existe nenhum desconto maior que 80%')

Output

existe pelo menos um desconto maior que 80%

Inaceitável

Quantas vezes é que o ciclo for é executado? É executado 10 vezes. (Porque há 10 descontos na lista e o corpo do ciclo for é executado uma vez para cada desconto. É isso que mostra o print que está comentado.)

Mas logo o terceiro desconto é maior que 80%. E portanto deveria para-se a execução aí. Continuar à procura nos restantes 7 descontos é demorar tempo desnecessário. É desperdiçar tempo. E se forem milhares ou milhões de descontos é muito tempo.

Solução

Usar um ciclo while. While quer dizer "enquanto" em português. Um ciclo while permite executar um bloco de código enquato uma condição for verdadeira.

Programa (no ficheiro exercicio_7_existe_desconto_grande_v2.py)

Escreva e execute este programa. Confirme o seu output.


lista_de_descontos = [10, 15, 95, 20, 10, 10, 30, 20, 10, 5]

desconto_grande = 80

índice = 0
existe = False
while existe == False and índice < len(lista_de_descontos):

    desconto = lista_de_descontos[índice]

    #print(índice, desconto) # descomente este print só para confirmar
                             # quantas vezes este ciclo while é executado
    if desconto > desconto_grande:
        existe = True

    índice = índice + 1

if existe == True:
    print('existe pelo menos um desconto maior que 80%')
else:
    print('não existe nenhum desconto maior que 80%')

Output

existe pelo menos um desconto maior que 80%
  • O que acontece se comentar a linha índice = índice + 1? O índice não avança. Testa-se apenas o primeiro desconto (no índice 0). E como o primeiro desconto não é maior que 80%, a variável existe nunca fica True e por isso entra-se num ciclo infinito!!! Que não faz nada útil. Cuidado!!! Se o computador estiver a executar um ciclo infinito que não faça nada útil desperdiça toda a energia que esteja a consumir. Para além de ficar bloqueado nessa execução.
